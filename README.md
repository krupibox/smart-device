# Smart Device
## Инструкция
> Установите
* Node.js v.10.15.3
* Gulp v4
* npm last version

> Команды для работы
* Установка - `npm i`
* Запуск локального сервера - `npm start`
* Сборка проекта без запуска локального сервера - `npm run-script build`
* Запуск тестирования на соответствие код-гайдам - `npm test`
